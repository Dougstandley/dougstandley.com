# dougstandley.com

Personal site. Built with Astro, deployed via Cloudflare Pages.

Essays live in `src/content/essays/` as markdown files. Set `status: published` in an essay's frontmatter to have it appear on the homepage.

## Local development

```
npm install
npm run dev
```

Then open http://localhost:4321 in your browser.

## Deploying

Push to `main` on GitHub. Cloudflare Pages rebuilds and deploys automatically.
